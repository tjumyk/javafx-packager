Javafx-packager

A GUI packaging tool to generate Single Runnable Jars for JavaFx2 programes

Confused about packaging JavaFx 2 Applications when using Eclipse and don't want to install a NetBeans?

You can check this simple online tool(Need Java & Browser Plugin Support) or download this project to run it locally.
